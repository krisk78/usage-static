Detailed Description

Class for unnamed arguments.

Values of unnamed arguments are passed directly through the command line. An unnamed argument still has a name. It is used to handle rules and display help. Instead of named arguments, this name has not to be passed through the command line. A single unnamed argument can accept many following values (i.e. a list of files).
